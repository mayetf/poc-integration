Notes d'intégration — POC 1C (JS SDK popup / side tab)

  • Setup : Remplacer YOUR_TOKEN et YOUR_DOMAIN dans le script ci-dessous
  • Modes : popup (overlay centré), sidetab (panneau latéral), popover (bulle attachée à un élément)
  • Déclencheurs : N'importe quel élément peut déclencher l'ouverture via SparrowLauncher.open()
  • Événements JS : onOpen, onClose, onSurveyComplete — visibles dans le log en bas à droite
  • Avantages : N'interrompt pas le scroll de la page, idéal pour CTA secondaires ou exit-intent
  • Limites : Le formulaire est dans un overlay — moins intégré visuellement qu'un inline, mais le theming reste complet

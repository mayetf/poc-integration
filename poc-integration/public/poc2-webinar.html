Notes d'intégration — POC 2 (Onomi / SpotMe)

  • Liste dynamique : GET /api/onomi/webinars — proxy vers l'API Onomi, filtré par statut published
  • Détail événement : GET /api/onomi/webinars/:id — récupère speakers, agenda, lien live
  • Inscription : POST /api/onomi/webinars/:id/register — crée l'attendee dans Onomi, retourne personal_link
  • Lien live personnalisé : personal_link dans la réponse d'inscription = URL unique par attendee pour rejoindre le stream Onomi
  • Lien live public : GET /api/onomi/webinars/:id/live-link — pour afficher un bouton "Rejoindre" quand l'événement est en cours
  • Statuts gérés : scheduled (inscription ouverte), live (bouton "Rejoindre maintenant" rouge), ended (bouton désactivé)
  • Email confirmation : send_confirmation_email: true dans la requête d'inscription → Onomi envoie son propre email de confirmation

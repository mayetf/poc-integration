POC 1B — JS SDK inline

Formulaire de contact

Le formulaire est chargé via le JS SDK SurveySparrow et rendu directement dans la page (inline). Le site contrôle les événements JS (start, answer, submit) pour déclencher des actions custom — analytics, pré-remplissage, redirections. Le theming peut être injecté via des CSS variables passées au SDK.

⬡ En attente du chargement du SDK SurveySparrow…

Notes d'intégration — POC 1B (JS SDK inline)

  • Setup : Remplacer YOUR_TOKEN dans le script ci-dessous par le token de votre survey (Share → Embed → JS)
  • Theming : Passer un objet customStyle au SDK (couleurs, polices, background) — surcharge le theme SurveySparrow
  • Événements JS : onSurveyStart, onQuestionAnswer, onSurveyComplete — visibles dans le log ci-dessus
  • Pré-remplissage : Passer des données via defaultAnswers (ex: email de l'utilisateur connecté)
  • Avantages : Contrôle JS total, auto-resize natif, intégration analytics, pré-remplissage
  • Limites : Dépend du JS client (SSR nécessite un traitement spécial), SDK à charger depuis CDN SurveySparrow

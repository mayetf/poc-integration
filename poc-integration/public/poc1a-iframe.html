POC 1A — iFrame embed

Formulaire de contact

Le formulaire ci-dessous est servi directement par SurveySparrow via un iframe. Le theming (couleurs, polices) est géré dans le Theme Builder SurveySparrow. Aucun JavaScript n'est requis côté site. Les réponses sont stockées dans SurveySparrow et un email de confirmation est envoyé automatiquement au répondant.

Notes d'intégration — POC 1A (iFrame)

  • Setup : Copier l'URL depuis Survey → Share → Embed (onglet "Website")
  • Theming : Géré via le Theme Builder dans SurveySparrow — couleurs, polices, background
  • Email confirmation : Configurer dans Survey → Settings → Email Notifications (template personnalisable)
  • Webhook : Configurer POST /webhook/surveysparrow dans Settings → Integrations → Webhooks pour déclencher des actions côté backend après soumission
  • Avantages : Zéro JS, logique conditionnelle 100% gérée par SurveySparrow, facile à updater sans redéploiement
  • Limites : Theming limité au theme builder, pas d'accès aux événements JS du formulaire, hauteur fixe ou auto-resize partiel
